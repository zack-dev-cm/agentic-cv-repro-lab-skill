# OpenClaw Research Lane

Use this reference when OpenClaw should help with public research or notebook-only work.

## Good uses

- public leaderboard pages
- public papers and supplementary material
- public GitHub repos and docs
- Colab or Kaggle notebook UIs that matter to the campaign

## Required outputs

- exact URL
- short note on why the source matters
- screenshot only when the visual state matters
- extracted facts copied into the campaign files

## Guardrails

- Prefer public pages and stable URLs.
- Do not depend on browser-only summaries when the same fact can be copied into a durable artifact.
- Do not treat browser LLM output as evidence without a reproducible benchmark result.
- Avoid private sessions, credentials, or personal browser state in public skill examples.
