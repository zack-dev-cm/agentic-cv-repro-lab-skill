# Public Research Lane

Use this reference when a SOTA campaign needs public research, leaderboard, or
repo evidence.

## Good uses

- public leaderboard pages
- public papers and supplementary material
- public GitHub repos and docs
- public notebook examples that matter to the campaign

## Required outputs

- exact URL
- short note on why the source matters
- screenshot reference only when the visible state matters
- extracted facts copied into the campaign files

## Guardrails

- Prefer public pages and stable URLs.
- Do not depend on page-only summaries when the same fact can be copied into a durable artifact.
- Do not treat generated prose as evidence without a reproducible benchmark result.
- Avoid private sessions, credentials, or personal account state in public skill examples.
